CSS Modal

Modals build out of pure CSS

Download CSS Modal

Build with pure CSS

CSS Modal is build out of pure CSS with some JavaScript sugar. This makes them perfectly accessible.

Optimized for mobile

The modals are designed with responsive web design in mind. They work on mobile and on desktop screen size.

Use as Sass plugin

You can use CSS Modal as Sass plugin and apply it to your custom classes. No need to understand all the code.

Examples

Show a text demo View a video demo

How to use

Sass

If you use Sass you can use the Sass file and include it into your project using @import "modal";

There are properties using Sass 3.2's placeholders. You can use them by calling @extend %modal; in your modal's class name.

CSS

If you are not into Sass you can still use Vanilla CSS.
